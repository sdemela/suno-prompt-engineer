FAQ

DOMANDE.

Cos'è uno Style prompt Suno e perché è importante? +
Il campo Style of Music in Suno è il parametro più potente per definire il suono del tuo brano. Un prompt ben costruito dà a Suno indicazioni precise su genere, energia, strumenti e texture. Un prompt vago produce risultati generici. Suno Prompt Engineer genera prompt calibrati per il comportamento specifico di ogni versione Suno.
Devo creare un account? +
No. Il tool è completamente anonimo. Un session ID viene generato automaticamente nel tuo browser. Puoi opzionalmente salvare la tua email per recuperare i crediti su altri dispositivi.
Cosa succede quando esaurisco le generazioni gratuite? +
Il piano gratuito si azzera ogni giorno a mezzanotte. Se ne hai bisogno di più, puoi acquistare crediti una volta — non scadono mai e sono legati al tuo session ID, utilizzabili su qualsiasi browser o dispositivo.
Qual è la differenza tra le versioni Suno? +
Ogni versione risponde a stili di prompt diversi. v3 richiede liste brevi di keyword, v4 e v4.5 funzionano meglio con descrittori stratificati, mentre v5 comprende frasi in linguaggio naturale e descrizioni sonore più ricche. Il nostro motore si adatta automaticamente.
Posso usare i prompt generati commercialmente? +
Sì. I prompt che generi sono tuoi. L'uso commerciale della musica generata da Suno dipende dai termini di servizio di Suno — consulta il loro sito per la policy aggiornata.
Cosa succede se cambio browser o dispositivo? +
I tuoi crediti sono legati al session ID. Inseriscilo nel campo "Ripristina crediti" su qualsiasi dispositivo per recuperarli. Salva il session ID o collegalo a un'email per sicurezza.
